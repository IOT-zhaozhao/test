# C++演讲比赛管理系统模板

#### Description
该项目提供了一个演讲比赛（或抽奖比赛）管理系统的demo，用于C++初学者进行学习。核心功能为设置演讲人数(抽奖人数)，通过系统抽签，选出冠亚季军（或中奖的前三人），提供菜单界面，交互性良好。
